## Why Vespera

<CardGrid stagger>
  <Card title="Framework-agnostic" icon="puzzle">
    The entire look lives in CSS variables on `.vsp-root`. Use it in React, Vue, Svelte,
    Angular — or plain HTML.
  </Card>
  <Card title="Themed by tokens" icon="setting">
    Swap theme, accent, density, and corners with data-attributes. No rebuild, no recompile.
  </Card>
  <Card title="Modular" icon="open-book">
    Ship the whole bundle, or import just the tokens / base / shell / components you need.
  </Card>
  <Card title="AI-consumable" icon="rocket">
    Built so coding agents can author and theme it correctly from a machine-readable manifest.
  </Card>
</CardGrid>

## See it in action

- **[Interactive playground](/vespera-ui/demo)** — every component on one page; click, toggle, and theme it live.

Vespera is being productized from a working prototype. APIs and class names may change

wrappers for **React**, **Vue**, **Svelte**, and **Angular** — see the
[Frameworks guide](/vespera-ui/guides/frameworks/) and the
[React components API](/vespera-ui/reference/react/).
:::
