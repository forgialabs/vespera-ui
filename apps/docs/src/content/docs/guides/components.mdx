Live examples you can interact with, grouped by category — the same grouping as the
[API reference](/vespera-ui/reference/react/), which has the full props of all 73 components.
Everything renders inside a themed `.vsp-root` — see
[Installation](/vespera-ui/guides/installation/) to set up.

Interactive demos with overlays (dropdowns, dialogs, date pickers…) run in an isolated frame,
so anything they pop open stays inside its preview.
