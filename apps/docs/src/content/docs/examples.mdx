## Build it with the packages

The admin console above is composed from the productized packages — here's the shape of it:

```tsx
import '@vespera-ui/css';
import { Card, CardHead, Button, Badge, Tabs } from '@vespera-ui/react';

function DashboardCard() {
  return (
    <div className="vsp-root" data-theme="dark">
      <Card>
        <CardHead title="Revenue" desc="Last 30 days" right={<Badge tone="pos" dot>+12%</Badge>} />
        <div className="card-pad">{/* charts, stats, tables… */}</div>
      </Card>
    </div>
  );
}
```

See [Installation](/vespera-ui/guides/installation/) to get started, the
[React components API](/vespera-ui/reference/react/) for every component, and the
[interactive playground](/vespera-ui/demo) to experiment live.
