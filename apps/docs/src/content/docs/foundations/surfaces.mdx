Vespera's "weightless, deep-space" feel comes from two things: a **layered background** that
recedes, and a **surface scale** that brings content forward — depth without heavy shadows.

## The background

`.vsp-root` paints `--bg` (a near-black `#080b14` in dark) and layers two radial gradients over
it: a cool glow top-right (`--bg-glow`) and a faint accent wash bottom-left. You get this for
free just by wrapping content in `.vsp-root` — no markup needed.

```css
.vsp-root {
  background:
    radial-gradient(900px 600px at 78% -8%, var(--bg-glow), transparent 60%),
    radial-gradient(700px 500px at -5% 110%, color-mix(in oklab, var(--accent) 10%, transparent), transparent 55%),
    var(--bg);
}
```

## Floating glow orbs

For a full-bleed app, add the optional **animated backdrop** — three blurred, slowly drifting
orbs (`position: fixed`, behind everything). This is the signature "weightless" motion. Drop one
empty element with three `<b>` children inside your `.vsp-root`:

```html
<div class="vsp-root" data-theme="dark">
  <div class="vsp-bg" aria-hidden="true"><b></b><b></b><b></b></div>
  <!-- your app -->
</div>
```

The orbs tint from `--bg-accent` (which tracks the accent) and respect
`prefers-reduced-motion`. Here it is live — note the slow drift behind the card:

## Surface elevation

Raised content steps up a three-level scale over `--bg`. Reach for the next surface as you nest,
rather than adding shadows:

| Token | Use for |
| --- | --- |
| `--bg` | The page itself (with the gradient above) |
| `--surface-1` | Cards and panels — the `.card` class uses this |
| `--surface-2` | Inputs, table rows, wells inside a card |
| `--surface-3` | Popovers, dropdown menus, hover states |

## Glass

Sticky bars and floating headers use `--glass` — a translucent surface meant to pair with
`backdrop-filter: blur()` so content scrolls softly underneath.

```css
.app-bar {
  background: var(--glass);
  backdrop-filter: blur(12px);
  border-bottom: 1px solid var(--border);
}
```
