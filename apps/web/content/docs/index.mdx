**Vespera** is a deep-space, weightless design system. The entire look lives in CSS variables on a
single `.vsp-root` element, so it themes with one attribute — no rebuild, no recompile — and works
in **React, Vue, Svelte, Angular, or plain HTML**.

This very site is built with Vespera itself (Next.js + Fumadocs) — the navbar, search, and every
demo are the real components.
