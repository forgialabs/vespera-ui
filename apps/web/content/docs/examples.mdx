## Build it with the packages

These templates are composed from the productized packages — here's the shape of it:

```tsx

function DashboardCard() {
  return (
    <div className="vsp-root" data-theme="dark">
      <Card>
        <CardHead title="Revenue" desc="Last 30 days" right={<Badge tone="pos" dot>+12%</Badge>} />
        <div className="card-pad">{/* charts, stats, tables… */}</div>
      </Card>
    </div>
  );
}
```
