# @vespera-ui/react

React components for the **Vespera** design system (the flagship framework package).

> 🚧 **Scaffolding.** Phase 2 ports the prototype's `.jsx` primitives into typed ESM
> components here. Styling comes from [`@vespera-ui/css`](../css); icons from
> [`@vespera-ui/icons`](../icons).

## Usage (planned)

```bash
pnpm add @vespera-ui/react @vespera-ui/css
```

```tsx
import '@vespera-ui/css';
import { Button } from '@vespera-ui/react';
```
