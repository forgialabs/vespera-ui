Add `FileDropzone` to the Vue and Svelte packages, matching React. Click to browse or drag-and-drop files; highlights on drag-over and emits the selected `File[]`. Supports `accept`, `multiple`, and a custom `hint`. (Angular ships the same as `vsp-file-dropzone`, version 0.14.0.)
