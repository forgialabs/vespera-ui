Add `CommandPalette` (⌘K-style command modal) to the Vue and Svelte packages, matching React. Controlled via `open` + `onclose`, takes `groups` of `CommandItem`s; supports live filtering, keyboard navigation (↑/↓/↵), Escape to close, and backdrop-dismiss. (Angular ships the same as `vsp-command-palette`, version 0.12.0.)
